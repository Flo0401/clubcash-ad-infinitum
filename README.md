# ClubCash – Ad Infinitum

Eine moderne Vereins-App für den Motorradclub Ad Infinitum.

## Geplante Funktionen

- 🔐 Login mit Benutzername und Passwort
- 💰 Guthabensystem
- 🍺 Getränkebuchung
- 📦 Lagerverwaltung
- 👑 Adminbereich
- 📊 Statistiken
- 📱 Installierbar auf iPhone und Android (PWA)

Entwicklung mit Unterstützung von ChatGPT.
