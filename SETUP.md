# Setup-Anleitung

## Voraussetzungen

Auf dem Laptop installieren:

- Node.js
- Git
- GitHub Konto

## Backend starten

```bash
cd backend
npm install
npm run dev
```

## Frontend starten

```bash
cd frontend
npm install
npm run dev
```

## Wichtig

Diese Phase-2-Version ist die Grundlage.
Die echte Datenbank wird im nächsten Schritt verbunden.
